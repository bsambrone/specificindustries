# Boom-Fun! Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the `boomfun` subdomain — a satirical 1961-vintage American toy catalog that cheerfully sells blasting caps and demolition toys to children. 12 pages (9 standard + 3 themed: Safety Corner, Club, Testimonials), 8 products, 4-person leadership team, commerce flow, and full portfolio integration.

**Architecture:** New site under `src/sites/boomfun/` following the established subdomain-site pattern. Reuses `CartProvider`, `AddToCartButton`, `Toast`, `CartButton`, `Hero`, `FeatureSection`, `Timeline`, and `TeamMember` from the shared component library. One dynamic route (`/products/[slug]`) plus one site-local component (`CatalogProductCard`) for hybrid 1960s catalog decorations. All routing resolves through the existing catch-all at `src/app/[[...slug]]/page.tsx`; no new App Router folders.

**Tech Stack:** Next.js 15 App Router, TypeScript, Tailwind CSS v4, shared `next/font` declarations. Product/exec imagery generated via OpenAI `gpt-image-1` through the `scripts/generate-*-images.ts` pattern. Testimonial portraits reuse the shared `/shared/testimonials/*.png` library (no new generation).

**Spec:** `docs/superpowers/specs/2026-04-24-boomfun-design.md`

**Verification approach:** The codebase has no unit-test suite for site pages. Each task verifies via `npx tsc --noEmit` (type safety), `npm run lint`, and (where reachable) `npm run dev` smoke checks at `localhost:3000/<path>?site=boomfun`.

**Voice rules (applied to all text content):**
- **Oblivious wholesome.** Site genuinely believes these are fine products. No self-aware winks.
- Register: 1961 Sears Christmas Wish Book, Daisy BB-gun catalog, Mattel-era toy copy.
- Safety language is sincere and cheerful — it makes the danger worse, never acknowledges it.
- The founder is Harland P. Crenshaw. The mascot is Sparky. The headquarters is Toledo, Ohio.
- No yellow/gold accent text on light backgrounds (portfolio convention).

import Image from "next/image"
import Link from "next/link"

interface CatalogProductCardProps {
  slug: string
  itemNumber: number
  stockNumber: string
  name: string
  tagline: string
  price: string
  image: string
  badge?: "NEW!" | "BESTSELLER!" | "SALE!"

export async function CatalogProductCard({
  slug,
  itemNumber,
  stockNumber,
  name,
  tagline,
  price,
  image,
  badge,
}: CatalogProductCardProps) {
  const siteHref = await getSiteHref()
  const resolvedHref = siteHref(`/products/${slug}`)

    <div className="relative border-2 border-primary/20 bg-background rounded-sm overflow-hidden group">

          }}
        >
          {badge}
        </div>
      )}

      {/* Product image */}
      <Link href={resolvedHref}>
        <div className="relative aspect-square bg-secondary/10">
          <Image src={image} alt={name} fill className="object-cover" />
        </div>
      </Link>

      {/* Body */}
      <div className="p-4">
        <div className="text-xs font-mono text-foreground/50 mb-1 uppercase tracking-widest">
          Stock No. {stockNumber}
        </div>

          <h3 className="text-xl font-heading font-bold text-primary leading-tight mb-1 uppercase">
